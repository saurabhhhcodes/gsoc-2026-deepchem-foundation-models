# Contributing to DeepChem DNA Foundation Models

Welcome to the **DeepChem biological foundation model** project! We are building the next generation of drug discovery tools.

## 🌳 Branching Strategy (GitFlow)
- `master`: Stable releases.
- `develop`: Main development bridge.
- `feat/feature-name`: New feature development.
- `fix/bug-name`: Critical bug fixes.

## 📝 Commit Standard (Conventional Commits)
Please follow the [Conventional Commits](https://www.conventionalcommits.org/) format:
- `feat: ...` (New features)
- `fix: ...` (Bug fixes)
- `docs: ...` (Documentation changes)
- `chore: ...` (Maintenance tasks)

## 🛠 CI/CD
Every pull request is automatically checked for code quality:
- **Linting**: Black, Flake8, Isort.
- **Testing**: PyTest for model loading and unit logic.
- **Benchmarks**: Automated tracking of model metrics.

## 🤝 Code of Conduct
Please be respectful and patient. We aim to build a diverse and inclusive community for researchers and AI engineers.
